# database/

Almacenar aquí los archivos de datos de clientes (`.csv`, `.xlsx`, `.xls`).

## Formato recomendado

| nombre | telefono | anotaciones |
|--------|----------|-------------|
| Juan Pérez | +5491112345678 | Cliente premium desde 2024 |
| María García | +5491187654321 | Consulta sobre producto X |

## ⚠️ IMPORTANTE

Estos archivos contienen datos sensibles de clientes y están excluidos del repositorio vía `.gitignore`.

- No se suben a GitHub
- No se comparten fuera del entorno local
- Mantener backup cifrado

Para agregar un archivo de ejemplo (sin datos reales) usar el sufijo `.sample`:
```
clientes.sample.csv
```
